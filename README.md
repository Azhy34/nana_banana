# Gemini Image Composer 🎨

Gemini Image Composer — это современное веб-приложение для генерации изображений с использованием передовых моделей Google Gemini. Приложение позволяет пользователям создавать детализированные изображения на основе текстовых описаний и опциональных референсных фотографий.

![Banner](https://github.com/user-attachments/assets/0aa67016-6eaf-458a-adb2-6e31a0763ed6)

## ✨ Основные возможности

*   **Генерация на базе Gemini**: Использует новейшие модели `gemini-3-pro-image` и `gemini-3.1-flash-image`.
*   **🎬 Анимация видео (Veo 3.1)**: Оживление сгенерированных 9:16 макетов с помощью модели Google Veo 3.1 (`veo-3.1-fast-generate-001`) для автоматического создания видео-пинов Pinterest с настраиваемыми траекториями камеры и seed-параметрами.
*   **Референсные изображения**: Возможность загрузить свое фото, которое послужит основой (композиция, стиль) для генерации нового изображения.
*   **Высокое разрешение**: Поддержка разрешений 1K, 2K и 4K (для Pro модели).
*   **Гибкие настройки**: Выбор соотношения сторон (1:1, 16:9, 9:16, 21:9 и др.).
*   **Безопасность**: Личные ключи Gemini/Replicate вводятся на клиенте, а тяжелые видеозадачи Veo обрабатываются через защищенные бэкенд-функции с использованием токена `GEMINI_VEO_API_KEY`.
*   **Современный UI**: Адаптивный интерфейс на базе Tailwind CSS с пошаговым мастером (Wizard) и Sleek Dark плеером для анимаций.

## 🚀 Быстрый старт

### Предварительные требования
*   [Node.js](https://nodejs.org/) (версия 18 или выше)
*   API-ключ Google Gemini (для работы в AI Studio)
*   Vercel CLI (для локального тестирования серверных API-функций)

### Настройка переменных окружения
Создайте файл `.env.local` в корне проекта и пропишите в него ключи:
```bash
GEMINI_VEO_API_KEY="ваш_платный_ключ_gemini_для_видео"
BLOB_READ_WRITE_TOKEN="токен_хранилища_vercel_blob"
```

### Установка и запуск

1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/Azhy34/nana_banana.git
    cd nana_banana
    ```

2.  Установите зависимости:
    ```bash
    npm install
    ```

3.  Запустите сервер с поддержкой API-эндпоинтов бэкенда:
    ```bash
    npx vercel dev
    ```

4.  Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## 🛠 Технологический стек

*   **Frontend**: React 19, TypeScript
*   **Сборка**: Vite
*   **Стили**: Tailwind CSS
*   **AI SDK**: @google/genai Node.js SDK
*   **Хранилище видео**: Vercel Blob Storage
*   **Трассировка и телеметрия**: Google Cloud Trace (через JSON-логи с полем `logging.googleapis.com/trace`)
