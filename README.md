# Gemini Image Composer 🎨

Gemini Image Composer — это современное веб-приложение для генерации изображений с использованием передовых моделей Google Gemini. Приложение позволяет пользователям создавать детализированные изображения на основе текстовых описаний и опциональных референсных фотографий.

![Banner](https://github.com/user-attachments/assets/0aa67016-6eaf-458a-adb2-6e31a0763ed6)

## ✨ Основные возможности

*   **Генерация на базе Gemini**: Использует новейшие модели `gemini-3-pro-image` и `gemini-3.1-flash-image`.
*   **Референсные изображения**: Возможность загрузить свое фото, которое послужит основой (композиция, стиль) для генерации нового изображения.
*   **Высокое разрешение**: Поддержка разрешений 1K, 2K и 4K (для Pro модели).
*   **Гибкие настройки**: Выбор соотношения сторон (1:1, 16:9, 9:16, 21:9 и др.).
*   **Безопасность**: API-ключ вводится пользователем в браузере и никогда не сохраняется на сервере.
*   **Современный UI**: Адаптивный интерфейс на базе Tailwind CSS с пошаговым мастером (Wizard).

## 🚀 Быстрый старт

### Предварительные требования
*   [Node.js](https://nodejs.org/) (версия 18 или выше)
*   API-ключ Google Gemini (можно получить в [Google AI Studio](https://aistudio.google.com/app/apikey))

### Установка

1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/Azhy34/nana_banana.git
    cd nana_banana
    ```

2.  Установите зависимости:
    ```bash
    npm install
    ```

3.  Запустите сервер для разработки:
    ```bash
    npm run dev
    ```

4.  Откройте [http://localhost:3000](http://localhost:3000) в браузере.
