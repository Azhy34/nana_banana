# BATCH РЕЖИМ — ПЛАН РАЗРАБОТКИ

## ШАГ 1 — Настройка батча (Setup Screen)

- [ ] Компонент `BatchSetup.tsx`
- [ ] Загрузка референс-фото обоев (переиспользовать `ImageUploader.tsx`)
- [ ] Выбор количества фото: кнопки [6] [9] [12] [15]
- [ ] Слайдер распределения форматов:
  - 9:16 — N штук (вертикаль, хорошо на телефонах)
  - 16:9 — N штук (горизонталь, широкие виды комнаты)
  - 1:1  — N штук (квадрат, главное фото листинга)
  - Сумма всегда = выбранное количество фото
- [ ] Кнопка [🎲 Сгенерировать промты] → переход к Шагу 2

---

## ШАГ 2 — Карточки промтов (Prompt Cards)

- [ ] Компонент `BatchPromptCards.tsx`
- [ ] Для каждого фото — отдельная карточка с тегами:
  - **Цвет**: Wales Grey / Barberry / Bumblebee / Sage / Cool Blue / Olivbraun / горчичный / какао
  - **Стиль**: Japandi 2.0 / Minimalismus 2.0 / Boho 2026 / Retro 2026 / Gender-Neutral
  - **Мебель**: Oliver Furniture / Paidi / Flexa / Geuther / Leander / Sirch / Pinolino
  - **Ключевой объект**: halbhohes Hochbett / модульная система / трансформер-кровать
  - **Освещение**: солнечный день / пасмурно / горит лампа / вечер / фонарики
  - **Ракурс**: фронтальный / из угла комнаты / сбоку
  - **Аксессуары**: 2-3 рандомных из списка (джут корзина, деревянные игрушки, льняные шторы, вельветовый плед, постер крупный формат)
  - **Формат**: берётся из распределения Шага 1
- [ ] Клик на любой тег → выпадающий список, выбрать другой вариант
- [ ] Кнопка [🔄] на карточке → полный перебросить рандом только этой карточки
- [ ] Кнопка [🗑️] → удалить карточку
- [ ] Кнопка [✏️ Редактировать промт] → открыть финальный текст промта в textarea, можно поправить вручную
- [ ] Кнопка [+ Добавить карточку] → добавить новую карточку с рандомом
- [ ] Кнопка [▶ Генерировать все] → переход к Шагу 3

---

## ШАГ 3 — Генерация (Progress Screen)

- [ ] Компонент `BatchProgress.tsx`
- [ ] Все запросы к Gemini идут **параллельно** (Promise.allSettled)
- [ ] Прогресс-бар для каждой карточки отдельно
- [ ] Статусы: ожидание / генерируется / готово / ошибка
- [ ] Если ошибка на одной карточке — остальные продолжают
- [ ] По мере готовности каждое фото сразу появляется в сетке результатов

---

## ШАГ 4 — Результаты (Results Grid)

- [ ] Компонент `BatchResults.tsx`
- [ ] Сетка фото (3 колонки на десктопе)
- [ ] Каждое фото:
  - [ ] Чекбокс выбора (для скачивания выборочно)
  - [ ] Под фото кнопки: [↓ Скачать] [↑ Upscaler] [✂️ Cropper] [🔄 Перегенерировать]
  - [ ] При [🔄] — показывает промт карточку только для этого фото, можно изменить и перегенерировать одно фото
- [ ] Кнопка внизу: [⬇️ Скачать выбранные (N)]
- [ ] Кнопка: [⬇️ Скачать все]
- [ ] Кнопка: [← Назад к промтам] — вернуться к Шагу 2 без потери карточек

---
